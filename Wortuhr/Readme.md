# Wortuhr
## 1. Idee
Die Idee war es, eine Wortuhr zu bauen und damit einen Eye-catcher im Wohnzimmer zu haben.
Schnell wurde ich auf folgends Github repository aufmerksam gemacht: https://github.com/ESPWortuhr/Multilayout-ESP-Wordclock.

## 2. Materialien
Um solch eine Wortuhr in DIY zu realiesiere zu können, wollte ich allerding nicht komplett alles selbst herstellen,
denn einige Komponenten sind schon für erschwingleiche Beträge im Netz zu bekommen.
